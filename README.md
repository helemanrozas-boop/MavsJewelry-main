# Mavs Jewelry

## Descripción

Este proyecto corresponde a la Evaluación Final de la asignatura Diseño Web.

Se desarrolló una página web para el emprendimiento Mavs Jewelry utilizando HTML5, CSS3 y Bootstrap.

## Objetivo

Crear un sitio web moderno, responsivo y funcional para promocionar los productos artesanales del emprendimiento.

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Funcionalidades

- Página de inicio.
- Quiénes Somos.
- Productos.
- Contacto.
- Redes Sociales.
- Diseño Responsive.
